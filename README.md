## Start

```
yarn
yarn test:unit
```

Note that [directly running jest will fail](https://github.com/vuejs/vue-cli/tree/dev/packages/%40vue/cli-plugin-unit-jest), and you should run tests with ```vue-cli-service test:unit```.
